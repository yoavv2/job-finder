## Accomplishments
- `agentRuns` (OBS-01): full run lifecycle (startedAt/finishedAt/status), work counters (processed/succeeded/failed), and cost telemetry (durationMs/tokens/estimatedCost) + error/metadata; indexed by `agent` for run-history queries.

## Decisions Made
- **estimated_cost as `real`:** integer would lose sub-cent precision for LLM cost telemetry; `real DEFAULT 0 NOT NULL` chosen.
- **status as typed text, not DB enum:** mirrors existing `jobs.status` convention (`$type<...>()` over `text`), keeping the type-safety in TS while the DB column stays plain text.
- **Composite (job_id, type) index on artifacts:** a single composite index serves both list-by-job and list-by-(job,type) lookups, avoiding a redundant single-column index.
